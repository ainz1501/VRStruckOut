VRコントローラーを用いてボールを投げ、正面のパネルに当てるゲームです。
[必要なもの]
・VRデバイス（ヘッドマウントディスプレイ（HDM）＋コントローラー）
・SteamVR
・Unity Hub（Editor Version=2021.3.4f）

[実行方法]
1. SteamVRからHDMとコントローラーを接続する
2. 本ファイルをUnityHubのプロジェクトとして追加
  3. "Editor Version"は 2021.3.4f を選択
4. 実行ボタンを押してスタート
   
[プレイ方法（Valve Indexコントローラーの場合）]
1. ボールに手を近づけ、トリガーボタンとトラックボタンを押すことでボールを掴む
2. トリガーボタンとトラックボタンから指を離すことでボールを投げることができる
3. キーボードのスペースキーを入力することでカゴにボールが補充される

[苦労・工夫した点]
・苦労した点
本来は、投げる直前までの手の速さや方向を用いて、ボールの飛ぶ方向やその強さを決める予定でしたが、
ボールを掴んだ瞬間以外は速度や角速度のパラメータを0に初期化していたため、ボールが全く飛ばなくなりました。
これを解消するためにボールを掴んでいる間は速度・角速度のパラメータを更新するようにしたのですが、それでも上手く動きませんでした。
授業の最後には自身の作品を発表・体験させる時間があったため、前方向に固定の強さで飛ばすことで解決しました。
・工夫した点
手に当たったボールの挙動が自然になるよう、手に設置したカプセルコリジョンや掴んだ際のボールの位置を手の形に沿うように配置しました。
また、ボールを掴むためにTriggerをオンにしたコリジョンを設定したのですが、このままではカゴや地面をすり抜けてしまうため、
Triggerをオフにしたコリジョンをその内側に設定することで、すり抜けずにボールに触れた判定を取得できるようにしました。
